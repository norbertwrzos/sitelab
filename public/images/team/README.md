# Team Member Images

This folder should contain team member photos.

## Recommended Image Specs:
- Format: WebP or JPEG
- Size: 400x400 pixels (1:1 square)
- Quality: 85%
- Max file size: 100KB

## Required Images:
- `alex-chen.jpg` - Founder & Lead Designer
- `sarah-johnson.jpg` - Full-Stack Developer
- `mike-williams.jpg` - UI/UX Designer

Fallback: The Avatar component will show initials if images are missing.
