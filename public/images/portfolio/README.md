# Portfolio Images

This folder should contain portfolio project images.

## Recommended Image Specs:
- Format: WebP or JPEG
- Size: 1200x800 pixels (3:2 aspect ratio)
- Quality: 80-85%
- Max file size: 200KB

## Required Images:
- `techgear.jpg` - E-commerce project
- `legal.jpg` - Legal services project
- `restaurant.jpg` - Restaurant project
- `healthcare.jpg` - Healthcare project
- `realestate.jpg` - Real estate project
- `fitness.jpg` - Fitness studio project

You can use placeholder images from:
- https://unsplash.com
- https://placehold.co
